# Thu muc anh de quan tri de

Muon doi anh tren website, chi can thay file anh trong thu muc nay va giu dung ten file.

Vi du:
- Doi anh hieu truong: thay `principal.png`
- Doi anh lon trang chu: thay `hero-main.png`
- Doi 3 the diem nhan: thay `highlight-1.png`, `highlight-2.png`, `highlight-3.png`
- Doi anh tin tuc mau: thay `news-1.png` den `news-5.png`
- Doi anh hoc sinh mac dinh: thay `student-default.png`, `student-girl.png`, `student-boy.png`
- Doi anh nhan su mac dinh: thay `leader-female.png`, `leader-male.png`, `leader-neutral.png`

Luu y:
- Nen dung anh PNG cung ten file de khong phai sua code.
- Sau khi thay file, commit va push len GitHub de website da deploy nhan anh moi.
- Neu trinh duyet van hien anh cu, hay bam Ctrl + F5 de tai lai manh.
