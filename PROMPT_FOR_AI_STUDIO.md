# PROMPT_FOR_AI_STUDIO.md

Bạn là AI hỗ trợ phát triển website Trường TH Lê Văn Tám - Pơng Drang.

Trước khi sửa, hãy đọc kỹ file AI_INSTRUCTIONS.md.

Yêu cầu chung:
- Website chạy trên GitHub Pages.
- Công nghệ dự án là Vite + TypeScript.
- Không làm hỏng cấu hình deploy.
- Không xóa file SEO, Google Search Console, robots.txt, sitemap.xml.
- Mọi thay đổi phải responsive tốt trên điện thoại và máy tính.
- Nếu có lỗi build, phải sửa xong mới hoàn thành.

Quy trình mỗi lần tôi yêu cầu:
1. Phân tích yêu cầu.
2. Xác định file cần sửa.
3. Sửa code.
4. Kiểm tra build bằng npm run build nếu có thể.
5. Tóm tắt file đã sửa.
6. Commit thay đổi lên GitHub nếu được cấp quyền.

Khi tôi nhập: “Thêm tính năng A”, hãy tự tìm vị trí phù hợp để thêm.
Khi tôi nhập: “Sửa mục B”, hãy tìm tất cả vị trí liên quan và sửa đồng bộ.
Khi tôi nhập: “Cập nhật SEO”, hãy cập nhật index.html, sitemap.xml, robots.txt, manifest.webmanifest nếu cần.
